# Job Titles Knowledge Base - Tech Roles

## Overview

Este arquivo contém mapeamento completo de cargos tech para detecção de seniority, matching de vagas e análise de currículos. Suporta títulos em Português (BR) e Inglês.

---

## Seniority Level Mapping

| Level | Code | Years (Typical) | Weight |
|-------|------|-----------------|--------|
| Intern/Trainee | 0 | 0 | -3 |
| Entry/Junior | 1 | 0-2 | -2 |
| Mid-Level | 2 | 2-4 | 0 |
| Senior | 3 | 4-7 | +3 |
| Lead/Staff | 4 | 6-10 | +4 |
| Principal/Specialist | 5 | 8-12 | +5 |
| Architect | 5 | 8-15 | +5 |
| Manager | 5 | 6-12 | +5 |
| Director | 6 | 10-15 | +6 |
| VP | 7 | 12-20 | +7 |
| C-Level | 8 | 15+ | +8 |

## Usage Examples

### Detect Seniority from Title

```python
def detect_seniority(title: str) -> dict:
    title_lower = title.lower()
    base_level = 2  # Default mid-level
    
    for keyword, modifier in SENIORITY_KEYWORDS.items():
        if keyword in title_lower:
            base_level += modifier
            break
    
    # Clamp level between 0 and 8
    level = max(0, min(8, base_level))
    
    return {
        "level": level,
        "level_name": LEVEL_NAMES[level],
        "title": title
    }
```

### Match Resume Title to Job Title

```python
def calculate_title_match(resume_title: str, job_title: str) -> float:
    resume_seniority = detect_seniority(resume_title)
    job_seniority = detect_seniority(job_title)
    
    # Check category match
    resume_category = detect_category(resume_title)
    job_category = detect_category(job_title)
    
    category_match = 1.0 if resume_category == job_category else 0.5
    
    # Check seniority match (within 1 level is acceptable)
    level_diff = abs(resume_seniority["level"] - job_seniority["level"])
    if level_diff == 0:
        seniority_match = 1.0
    elif level_diff == 1:
        seniority_match = 0.8
    elif level_diff == 2:
        seniority_match = 0.5
    else:
        seniority_match = 0.2
    
    return (category_match * 0.6) + (seniority_match * 0.4)
```
